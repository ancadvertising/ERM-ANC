# تقرير التحقق

تاريخ التحقق: 2026-07-21

## النتائج

- نجح فحص JavaScript syntax لعدد 42 ملفًا من ملفات الواجهة وGoogle Apps Script.
- نجح تحميل صفحة الواجهة وكل ملفات CSS وJavaScript عبر خادم HTTP محلي برمز 200.
- نجح اختبار دخول بوابات الإدارة والموظف والعميل في الوضع المحلي المعزول.
- حساب الإدارة أعاد 14 عنصرًا في القائمة الرئيسية.
- حساب الموظف أعاد بوابة الموظف فقط.
- حساب العميل أعاد بوابة العميل فقط.
- نجح تحميل لوحة الإدارة وصفحة الحسابات ببيانات الاختبار.
- لا يوجد أي استخدام لـ `google.script.run` داخل الواجهة.
- يوجد تعريف واحد فقط لـ `doGet` وتعريف واحد فقط لـ `doPost`.
- manifest صالح بصيغة JSON ومنطقة الوقت `Africa/Cairo`.
- لا توجد محارف Unicode Replacement Character.
- لا توجد سلاسل قابلة لعكس mojibake في ملفات التسليم.
- روابط ملفات `config.js` و`api.js` و`ui.js` و`views.js` و`app.js` موجودة ومرتبة.

## نطاق الاختبار

تم اختبار طبقة الواجهة ببيانات Demo لا تعمل إلا على localhost. تشغيل خدمات Google الفعلية يحتاج نشر مجلد `backend` في حساب Google وتشغيل `setupProduction()`.

تعذر تشغيل المتصفح المدمج آليًا بسبب فشل مشغل Windows sandbox في بدء جلسة المتصفح. تم التعويض بفحوص HTTP وsyntax وUTF-8 واختبارات دخول ومسارات البوابات. يوصى بعد النشر بفتح رابط GitHub Pages مرة على هاتف ومرة على كمبيوتر للتحقق من الشكل النهائي مع بيانات المؤسسة الحقيقية.
